Licença e uso:
- A maioria dos brasões e logotipos de clubes são material com direitos reservados. Estas cópias disponíveis no Wikimedia são fornecidas sob critérios de uso não-livre (fair use) e/ou uploads com metadados. Verifique a página do arquivo no Wikimedia para detalhes de licença antes de republicar para fins comerciais.


Como usar:
1. Crie a pasta `assets/logos/` (já presente neste projeto) caso queira hospedar os arquivos localmente.
2. Baixe cada arquivo e salve com o nome sugerido.
3. Atualize os `src` em `index.html` para apontar para `assets/logos/<nome>` caso você prefira servir localmente.

Observação: neste momento o site foi atualizado para utilizar os arquivos diretamente do Wikimedia (hotlink) para exibir os logos reais sem necessidade de download local. Se quiser que eu baixe e incorpore os arquivos ao repositório, responda "SIM, baixe" e eu os adiciono.
